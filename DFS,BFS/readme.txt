1. 최댓값을 -100000000, 최솟값을 100000000으로 설정해준다. 이유는 추후 최댓값, 최솟값이 나올경우 갱신해 주기 위해 초기값 범위를 넓게 잡기위해서이다.
2. 들어가는 정수의 개수를 입력받고, 개수만큼 배열(number[])의 크기를 설정해준뒤, 배열에 숫자 입력값들을 넣어준다.
3. 마찬가지로 연산자를 입력받는 배열을 만들고, 안에 연산자를 입력한다.
4. 배열의 시작인 number[0]부터 dfs를 돌린다.
5. dfs를 반복하는 횟수는 N-1번 반복하게 되고, 결과값과 최솟값, 최대값을 비교하여 갱신해준다.
6. dfs의 경우, 연산자의 배열에서 연산자를 꺼내 연산을 실행하고 횟수를 1씩 증가시킨다. 더이상 연산자가 존재하지 않을 경우까지 재귀를 통해 dfs과정을 반복한다.
7. 모든 과정을 마친 후, 최댓값과 최솟값을 출력한다.
